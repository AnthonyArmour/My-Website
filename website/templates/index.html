Transformer for Machine Translation

Machine Translation is a sub-field of computational linguistics that uses software to translate text or speech from one language to another. A transformer is a deep learning model that uses the mechanism of self-attention, differentially weighting the significance of each part of it's input. Like Recurrent Neural Networks, transformers are designed to handle sequential input data. However, unlike RNNs, the transformer's attention mechanism provides context for any position in the input sequence, making it unnecessary to process the data in order. Transformers are primarily used in the fields of natural language processing and computer vision. In this project, I created and trained a transformer model for translating Portuguese text to English text.

Bert Question Answering Bot

Question answering is a computer science discipline within the fields of information retrieval and natural language processing, used for building systems that automatically answer questions posed by humans. Question Answering systems construct its answers by querying a structured database of information or knowledge, known as a knowledge base. Bidirectional Encoder Representations from Transformers (BERT) is a transformer-based machine learning technique for natural language processing. In this project, I used a semantic search algorithm for parsing a knowledge base and retrieving the documents that are the most relevant to the question. Then use a BERT language model to generate the answer to the initial question, using the most relevant document.

Deep Q-Learning Agent for Atari Breakout

Q-learning is a model-free reinforcement learning algorithm to learn the value of an action in a particular state. Q-learning finds an optimal policy by maximizing the expected value of the total reward over successive steps, starting from a specific state. Deep Q-learning uses a neural network as a non-linear approximator of the action-value function, giving the agent generalizing characteristics and the ability to model large state spaces. For this project, I trained a convolutional neural network to approximate the action-value function of the game Atari Breakout.

Bitcoin Closing Price Model Using LSTM Recurrent Neural Network

Recurrent Neural Networks (RNNs) are a robust type of neural network that gain their power by including an internal memory state of input history to help make predictions on future time steps. The meaningful difference between RNNs and traditional feed-forward neural networks is the way that they channel information. In an RNN the information cycles through a loop. When it makes a prediction, it considers the current input and observations from previously received inputs. For this project, I trained a RNN on Bitcoin time series data for predicting the closing price of Bitcoin at the end of a specific time window. Clustering and Mixture Models

k-means clustering is a method of vector quantization, originally from signal processing, that aims to partition n observations into k clusters in which each observation belongs to the cluster with the nearest mean and minimize the within-cluster variances. Formally, mixture models correspond to the mixture distribution that represents the probability distribution of observations in the overall population. A multivariate Gaussian mixture model is used to cluster the feature data into k number of groups where k represents each state of the machine. These algorithms are unsupervised techniques grouping data in a fashion that minimizes the variance between points. For this project, I implement the k-means clusting and Gaussian mixture model algorithms from scratch in Python.

Gaussian Processes and Bayesian Optimization

Used to model a function given a small amount of data points, a Gaussian process is a stochastic process (a collection of random variables indexed by time or space), such that every finite collection of those random variables has a multivariate normal distribution. Gaussian processes can be seen as an infinite-dimensional generalization of multivariate normal distributions. Gaussian processes fit the data by getting the mean over distributions and define uncertainty along the model by the variance along those sampled distributions. Bayesian optimization is a sequential design strategy for global optimization of black-box functions that does not assume any functional forms. It is usually employed to optimize expensive-to-evaluate functions. It takes a model like a gaussian process to define a posterior and works by finding the sample location that maximizes the expected improvement of the objective function. For this project, I created a Gaussian process from scratch and use it for Bayesian optimization to model a blackbox function.
